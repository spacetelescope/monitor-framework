Monitoring Framework
====================

Light-weight framework designed to make writing new instrument monitors quick and easy.

To install, clone the repository and use ``pip``::

   cd monitoring
   pip install .

Or execute ``setup.py`` directly::

   cd monitoring
   python setup.py
